# ItalianPizza App

A website for ordering pizza online

## Running Locally

1. clone this repo
2. cd client
3. npm i --force
4. npm start
   
Then open another new terminal 
1. cd server
2. npm i --force
3. npm start

## Tech Stack (MEAN STACK)

1. React.js
2. Advanced Topics in React(Redux,Redux Thunk)
3. Node.jS/express.jS
4. MongoDB/Mongoose
5. Heroku (hosting)
